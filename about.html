Project Journey

How This Project Grew

What started as a simple product listing page became a full e-commerce system — complete with authentication, cart, orders and live status tracking.

See It in Action
PHASE 01
Concept & Design
Planned the site structure, color palette, dark theme, and page layout before writing a single line of code.
PHASE 02
Core Pages Built
Built index.html, products.html and product.html with responsive Bootstrap 5 grid and dark theme CSS.
PHASE 03
Firebase Integration
Added Firebase Auth for login/register, Firestore for products, orders, cart and reviews.
PHASE 04
Order System
Built a Flipkart-style order tracker with live status updates — pending → confirmed → packed → delivered.
PHASE 05
Complete & Portfolio Ready
10+ pages, fully responsive, Firebase-powered — deployed as a portfolio project showcasing full-stack frontend skills.
Features Built

What This Project Demonstrates

Every feature was built from scratch — no templates, no page builders.

Firebase Auth
Login, register, logout and protected pages — all handled with Firebase Authentication.
Firestore Database
Products, orders, cart and reviews stored and fetched in real time from Cloud Firestore.
Order Tracking
Flipkart-style order status tracker with live updates — customers see status change in real time.
Fully Responsive
Every page works perfectly on mobile, tablet and desktop using Bootstrap 5 responsive grid.
Project Note

Simulated Business Concept

The cold storage business, products, prices and customer reviews on this site are fictional — created purely to demonstrate a realistic e-commerce experience. All technical features including authentication, cart and order management are fully functional.
